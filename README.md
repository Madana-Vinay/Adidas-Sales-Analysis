# Adidas-Sales-Analysis
Analyzed 9,648 Adidas US sales transactions (2020–2021) in Power BI. Built an interactive dashboard tracking $900M revenue, $332M operating profit &amp; 42% operating margin across 6 retailers, 5 US regions, 6 product categories &amp; 3 sales channels. Revealed seasonal trends, regional gaps &amp; top-performing retailer insights to drive strategy.
